# tic-tac-toe

### Description

A php-cli Tic-Tac-Toe simple two players game.

### Requirements

1. `php >= 8`
2. `Docker`

### Installation

The project has a `Makefile` which will take care of everything required for installation.

Run `make all` will do all the magic for you.
